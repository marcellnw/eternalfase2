# eternalfase2
Fase 2 EternalSMP resmi dibuka dengan sistem pendaftaran baru, mekanisme Kerajaan dan Party yang lebih terstruktur, serta quest, leveling, dan ADC yang diperbarui. Tahap ini dirancang untuk menciptakan progres yang kompetitif, tertata, dan berkelanjutan, sekaligus mempersiapkan pondasi dunia menuju Season 14.
